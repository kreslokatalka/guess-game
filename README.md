# Угадай число

Консольная игра с подсказками «горячо/тепло/холодно». Можно настроить максимальное значение и количество попыток. Результаты сохраняются в `results.json`.

## Зависимости

- Go 1.26+
- `github.com/fatih/color`

## Установка

```bash
git clone https://github.com/kreslokatalka/guess-game.git
cd guess-game
go run guess.go
